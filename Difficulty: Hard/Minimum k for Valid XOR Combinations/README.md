<h2><a href="https://www.geeksforgeeks.org/problems/kx-subarrays--132103/1?page=3&difficulty=Hard&status=unsolved&sortBy=submissions">Minimum k for Valid XOR Combinations</a></h2><h3>Difficulty Level : Difficulty: Hard</h3><hr><div class="problems_problem_content__Xm_eO"><p><span id="docs-internal-guid-de647ee7-7fff-caa5-4ed1-650502814ea9"></span></p>
<p><span style="font-size: 14pt;">You are given a sequence of playing cards with numbers written on them, represented by an array <strong>arr[]</strong>. You can select any <strong>contiguous </strong>sequence of cards(cards must be consecutive) and compute the bitwise <strong>XOR </strong>of the numbers in the selected sequence. A sequence is considered <strong>valid </strong>if the XOR of its numbers is <strong>at most k</strong>.</span></p>
<p><span style="font-size: 14pt;">The task is to find the <strong>smallest value of k </strong>such that there are <strong>at least</strong> <strong>x </strong>valid sequences.</span></p>
<p><span style="font-size: 18px;"><strong>Examples :</strong></span></p>
<pre><span style="font-size: 18px;"><strong>Input: </strong>x = 3, arr = [1, 2, 3]<br><strong>Output: </strong></span><span style="font-size: 18px;">1<br><strong>Explanation:</strong> If k = 0, there is 1 subarray with xor ≤ 0. </span><span style="font-size: 18px;">If k = 1, there are 3 subarrays with xor ≤ 1, hence the minimum value of k is 1.</span></pre>
<pre><span style="font-size: 18px;"><strong>Input: </strong>x = 7, arr = [1, 6, 1, 4]<br><strong>Output:</strong> 6<br><strong>Explanation:</strong> If k = 5, there are 6 subarrays with xor ≤ 5, which is lesser than x. </span><span style="font-size: 18px;">If k = 6, there are 8 subarrays with xor ≤ 6, which is greater than equal to x.<br><img src="https://media.geeksforgeeks.org/img-practice/prod/addEditProblem/878974/Web/Other/blobid1_1732085339.png" alt="" width="364" height="500"></span><br><br><span style="font-size: 18px;">Hence, k = 6 is the minimum value of k.</span></pre>
<p><br><strong>Constraints:<br></strong><span style="font-size: 14pt;">1 ≤ arr.size() ≤ 10<sup>5<br></sup>1 ≤ x ≤ ((arr.size())*(arr.size()+1))/2</span><br><span style="font-size: 14pt;">1 ≤ arr[i] ≤ 10<sup>4</sup></span></p></div><br><p><span style=font-size:18px><strong>Topic Tags : </strong><br><code>Binary Search</code>&nbsp;<code>Trie</code>&nbsp;

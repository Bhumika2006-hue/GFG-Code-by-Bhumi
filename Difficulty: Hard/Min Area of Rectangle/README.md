<h2><a href="https://www.geeksforgeeks.org/problems/min-area-of-rectangle--145846/1?page=3&difficulty=Hard&status=unsolved&sortBy=submissions">Min Area of Rectangle</a></h2><h3>Difficulty Level : Difficulty: Hard</h3><hr><div class="problems_problem_content__Xm_eO"><p dir="ltr" style="line-height: 1.38; margin-top: 12pt; margin-bottom: 12pt;"><span style="font-size: 14pt; font-family: Roboto, sans-serif; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">Geek is given a string <strong>path</strong> consisting of the characters <strong>'U', 'D', 'L'</strong> and <strong>'R'</strong>. These characters represent movements in a <strong>2D grid, </strong>where the <strong>top left corner</strong> is <strong>(0, 0).</strong></span></p>
<ul>
<li dir="ltr" style="line-height: 1.38;" role="presentation"><span style="font-size: 14pt; font-family: Roboto, sans-serif; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;"><strong>'D'</strong> : Move one cell down.</span></li>
<li dir="ltr" style="line-height: 1.38;" role="presentation"><span style="font-size: 14pt; font-family: Roboto, sans-serif; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;"><strong>'U'</strong> : Move one cell up.</span></li>
<li dir="ltr" style="line-height: 1.38;" role="presentation"><span style="font-size: 14pt; font-family: Roboto, sans-serif; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;"><strong>'R'</strong> : Move one cell right.</span></li>
<li dir="ltr" style="line-height: 1.38;" role="presentation"><span style="font-size: 14pt; font-family: Roboto, sans-serif; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;"><strong>'L'</strong> : Move one cell left.</span></li>
</ul>
<p dir="ltr" style="line-height: 1.38; margin-top: 12pt; margin-bottom: 12pt;"><span style="font-size: 14pt; font-family: Roboto, sans-serif; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">Initially, Geek is at cell <strong>(0, 0)</strong> in a rectangle grid of size <strong>n * m</strong> (where n represents the number of rows and m represents the number of columns).</span></p>
<p dir="ltr" style="line-height: 1.38; margin-top: 12pt; margin-bottom: 12pt;"><span style="font-size: 14pt; font-family: Roboto, sans-serif; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">Note that n and m are not provided as inputs and must be <strong>determined</strong> by you to satisfy the <strong>conditions</strong> described : -</span></p>
<ul>
<li dir="ltr" style="line-height: 1.38;"><span style="font-size: 14pt; font-family: Roboto, sans-serif; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">Geek must follow the movements in the <strong>path</strong> string in order. </span></li>
<li dir="ltr" style="line-height: 1.38;"><span style="font-size: 14pt; font-family: Roboto, sans-serif; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">However, if a movement causes Geek to step outside the boundaries of the rectangle grid, he remains in his current cell instead of moving.</span></li>
</ul>
<p dir="ltr" style="line-height: 1.38; margin-top: 12pt; margin-bottom: 12pt;"><span style="font-size: 14pt; font-family: Roboto, sans-serif; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">Given the final cell <strong>(fr, fc)</strong> that geek must reach by following the entire path, determine the minimum possible area of the rectangle, <strong>n * m</strong> that allows geek to complete the path and reach (fr, fc) at the end. If it is <strong>not possible</strong> for Geek to <strong>reach the target cell (fr, fc)</strong> using the given path, return <strong>-1</strong>.</span></p>
<p><span style="font-size: 18px;"><strong>Examples:</strong></span></p>
<pre><span style="font-size: 18px;"><strong>Input: </strong>path = "DDRR", fr = 1, fc = 2<br><strong>Output: </strong>6</span><span style="font-size: 18px;"><br><strong>Explanation:</strong> <br><img src="https://media.geeksforgeeks.org/img-practice/prod/addEditProblem/881445/Web/Other/blobid0_1733398571.jpg" width="507" height="337"><br>For n = 2 and m = 3, we get a minimum area of 2 * 3 = 6.</span></pre>
<pre><span style="font-size: 18px;"><strong>Input: </strong>path = "DDRRLUDUD", fr = 99, fc = 99<br><strong>Output:</strong> -1<br><strong>Explanation:</strong> It is not possible to reach (99, 99) by following the path with any value of n and m. Hence, we return -1.<br></span></pre>
<pre><span style="font-size: 14pt;"><strong>Input: </strong>path = "DURR", </span><span style="font-size: 14pt;">fr = 0, </span><span style="font-size: 14pt;">fc = 1</span><br><span style="font-size: 14pt;"><strong>Output: </strong>2</span><br><span style="font-size: 14pt;"><strong>Explanation:</strong> </span><br><span style="font-size: 14pt;">Geek <strong>starts</strong> from <strong>(0,0)</strong>, assuming n = 1 and m = 2.</span><br><span style="font-size: 14pt;">The Path goes as follows:</span><br><span style="font-size: 14pt;"><strong>(0,0) -&gt; (0,0) -&gt; (0,0) -&gt; (0,1) -&gt; (0,1)</strong></span><br><span style="font-size: 14pt;">For n = 1 and m = 2, we get a minimum area of <strong>1 * 2 = 2.</strong></span></pre>
<p dir="ltr" style="line-height: 1.38; margin-top: 12pt; margin-bottom: 12pt;"><span style="font-size: 14pt;"><span style="font-family: Roboto, sans-serif; color: #000000; background-color: transparent; font-weight: bold; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">Constraints:</span></span></p>
<ul>
<li dir="ltr" style="line-height: 1.38;"><span style="font-size: 14pt;"><span style="font-family: Roboto, sans-serif; color: #000000; background-color: transparent; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">1 ≤ | path | ≤ 2*10<sup>5</sup></span></span></li>
<li dir="ltr" style="line-height: 1.38;"><span style="font-size: 14pt;"><span style="font-family: Roboto, sans-serif; color: #000000; background-color: transparent; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">0 ≤ fr, fc &lt; 10<sup>5</sup></span></span></li>
<li dir="ltr" style="line-height: 1.38;"><span style="font-size: 14pt;"><span style="font-family: Roboto, sans-serif; color: #000000; background-color: transparent; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">path consists only of 'D', 'U', 'R' and 'L'</span></span></li>
